How This Blog is Made Part I

March 15, 2017
This blog is made of one philosophy resting on four pillars: it should cost exactly nothing and all the software you need are the following. The R programming language for data analysis. The R Markdown language for content generation and typesetting. Blogdown in conjunction with Hugo to generate the web pages. Github-Pages for webhosting. This post will mainly touch on 1 and 2. Part II will focus on 3 and 4 so you can get your own blog up and running if you want to follow in my footsteps.
